   1.声明 `bool cmp(const type &a, const type &b)`函数,例如：

   ```C++
   bool cmp(const STU &a, const STU &b)
   {
       if (a.sum != b.sum)
           return a.sum > b.sum;
       else if (a.cn != b.cn)
           return a.cn > b.cn;
       else
           return a.num < b.num;
   }
   ```
